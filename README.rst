========
Diskette
========

Export and import Django application data and storages.

Application datas are using Django fixtures in JSON.

Storages are not Django storages, only a set of directories to copy.


Dependancies
************

* `Python`_>=3.9;
* `Django`_>=4.0,<5.0;
* `django-sendfile2`_>=0.7.0;
